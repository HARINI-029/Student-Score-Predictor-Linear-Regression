# Student Score Predictor using Linear Regression

## Overview

The Student Score Predictor is a Machine Learning project that predicts a student's exam score based on the number of hours studied. The project uses the Linear Regression algorithm to identify the relationship between study time and academic performance.

This project demonstrates the complete machine learning workflow, including data preprocessing, model training, evaluation, and prediction.

---

## Objectives

* Analyze the relationship between study hours and exam scores.
* Build a Linear Regression model for prediction.
* Visualize the data and regression line.
* Evaluate model performance using standard metrics.

---

## Dataset

The dataset contains:

* Hours Studied
* Student Scores

The dataset used in this project is included in the repository.

## Machine Learning Workflow

### 1. Data Collection

* Loaded the dataset using Pandas.

### 2. Data Preprocessing

* Checked for missing values.
* Prepared features and target variables.

### 3. Data Visualization

* Created scatter plots to visualize the relationship between study hours and scores.

### 4. Model Training

* Split data into training and testing sets.
* Trained a Linear Regression model.

### 5. Prediction

* Predicted student scores based on study hours.

### 6. Evaluation

* Compared actual and predicted values.
* Measured model accuracy and performance.
